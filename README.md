## Deskripsi

Sistem Manajemen Perpustakaan merupakan aplikasi berbasis **Object-Oriented Programming (OOP)** yang dirancang untuk membantu proses pengelolaan perpustakaan secara lebih efektif dan terstruktur.

Sistem ini memungkinkan admin untuk mengelola data buku, data anggota, proses peminjaman, hingga pengembalian buku melalui aplikasi berbasis console.

# Latar Belakang

Beberapa permasalahan yang melatarbelakangi pembuatan sistem ini antara lain:

- Pengelolaan perpustakaan secara manual masih sering menimbulkan kesalahan pencatatan.
- Proses pencarian dan pengelolaan data masih kurang efisien.
- Banyak perpustakaan kecil yang belum memiliki sistem digital.
- Implementasi konsep Object-Oriented Programming perlu diterapkan pada studi kasus nyata agar mahasiswa lebih memahami penggunaannya.

---

# Tujuan

Project ini bertujuan untuk:

- Merancang sistem manajemen perpustakaan berbasis OOP.
- Mengelola data buku dan anggota.
- Mengelola transaksi peminjaman dan pengembalian buku.
- Mengimplementasikan konsep OOP secara nyata.

---

# Manfaat

- Membantu memahami implementasi Object-Oriented Programming.
- Memberikan contoh aplikasi berbasis OOP.
- Menjadi dasar pengembangan aplikasi berbasis GUI maupun Database.

---

# Konsep OOP yang Digunakan

Project ini menerapkan empat pilar utama Object-Oriented Programming.

## Class & Object

Beberapa class utama:

- User
- Admin
- Buku
- Anggota
- Perpustakaan

## Encapsulation

- Menggunakan access modifier
- Getter dan Setter

## Inheritance

Class:

- Admin
- Anggota

mewarisi class:

- User

## Polymorphism

Method:

```cpp
tampil_info()
```

diimplementasikan berbeda pada masing-masing class.
